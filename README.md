# Whispers of Orion
Is a p5.js project that connects math and poetry through interactive constellations. When users hover over the stars, short verses appear. The "decipher" button lets viewers see the whole poem at once, like revealing a hidden message.

This project mixes math structure and languages through functions and conditions to turn coordinates into meaning. Its about finding balance between technical design and creative expression.
